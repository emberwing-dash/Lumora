## 📖 Overview

Lumora is a VR-based educational platform that transforms traditional learning into an interactive and immersive experience.

Players explore a mysterious science temple where core academic concepts are disrupted. To restore balance, they must solve hands-on challenges and puzzles.

---

## ✨ Features

### 🧠 Concept-Based Learning
- Physics → Laws of Motion  
- Mathematics → Linear Equations  
- Chemistry → Basic Chemical Experiments  

### 🕶️ Immersive VR Interaction
- Real-world object manipulation  
- Physics-driven gameplay  
- Interactive environments  

### 🤖 AI-Powered Guidance
- Integrated using Convai  
- AI teacher provides hints and explanations  
- Adaptive responses based on player actions  

### 🧩 Puzzle-Driven Progression
- Learn by solving challenges  
- Each room focuses on a concept  
- Progression based on understanding  

---

## 🗺️ Game Flow

Start Room → Learning Chambers → Concept Challenges → Final Core
